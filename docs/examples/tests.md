# Check if tests exist

## Example `rivi.rules.yaml`

```yaml
rules:
    has-tests:
        condition:
          order: 8
          files:
            patterns:
              - "\\/.*\\/[a-zA-Z0-9]*Test\\.(java|scala)$"
          skip-if-labeled:
            - ok-no-tests
        labeler:
          label: test-exists
          remove: no-tests
    
    no-tests:
      condition:
        order: 9
        files:
            extensions:
              - ".java"
              - ".scala"
        skip-if-labeled:
          - test-exists
          - ok-no-tests
          - no-tests
      labeler:
        label: no-tests
      commenter:
        comment: "You probably forgot your tests, right?"
```

**Note** The labels `ok-no-tests`, `no-tests`, `test-exists` must exist in the repository settings
