# Review changes to build files

## Example `rivi.rules.yaml`

```yaml
rules:
    build:
      condition:
        files:
          patterns:
            - "gradle.*"
            - "wrapper/"
          extensions:
            - ".gradle"
      labeler:
        label: system/build
      autoassign:
        role: devops
        
roles:
    devops:
      - dev1
      - ops2
      - ops3
```

**Note** The label `system/build` must exists in the repository settings  
**Note** The users `dev1`,`ops2`, and `ops3` must have repository read permissions
