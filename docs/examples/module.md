# Changes to a module

## Example `.rivi.rules.yaml`

```yaml
rules:
    engine-module:
      condition:
        files:
          patterns:
            - "engine/"
      labeler:
        label: module/engine
```

**Note** The label `module/engine` must exists in the repository settings
