# Warn if committing binaries

## Example `rivi.rules.yaml`

```yaml
rules:
    binary:
        condition:
          files:
            extensions:
              - ".jar"
              - ".zip"
              - ".tar"
              - ".gz"
              - ".bz2"
              - ".z"
              - ".tgz"
              - ".deb"
              - ".rpm"
        labeler:
          label: binary
        commenter:
          comment: "Please don't push binaries to our repository"
```

**Note** The label `binary` must exists in the repository settings
